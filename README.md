## Wolf3d
* *A wolf3d like game, made with raycasting*

## demo

## Built with
C language, SDL2 library, Raycasting method

## Installing
* Git clone http://github.com/jsauron/doom.git
* Compile it with `make`

## Make it work
Run `./wolf3d [map]`

## Make a map
First line : width height</br>
Bloc 1 : wall</br>
Bloc 0 : void</br>
Bloc 2 : spawn

## Map Exemple

```
4 4
1 1 1 1
1 0 0 1
1 2 0 1
1 1 1 1
```
